# Sunnyside Pet Care

Scaffolded demo project with a Spring Boot backend and a Vite + React frontend.

Backend (Spring Boot, H2 in-memory for dev):

From project root:

```powershell
cd C:\Users\tonyf\Documents\test\sunnyside-pet-care\backend
..\..\mvnw.cmd spring-boot:run  # or 'mvn spring-boot:run' if you have Maven
```

Frontend (Vite + React):

```powershell
cd C:\Users\tonyf\Documents\test\sunnyside-pet-care\frontend
npm install
npm run dev
```

API health: http://localhost:8080/api/health
Frontend: http://localhost:3000/
